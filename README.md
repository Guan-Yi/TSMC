# The Same Month in Calendar (TSMC)

## Usage
```$ python3 tsmc.py <target year> <interval to find>```

## Example
```$ python3 tsmc.py 2018 20```

Result:

```
January is same as: 
2001
2007
2018
2024
2029
2035
February is same as: 
2001
2007
2018
2029
2035
Other months are same as: 
2001
2007
2012
2018
2029
2035
```
